# Basestyles

Better default styles for modern browsers.

## Prerequisites

- [Node.js](https://nodejs.org)
- [npm](https://www.npmjs.com)

## Installation

1. Make sure your system has everything listed in the prerequisites section
2. Clone the repo
3. Run `npm install`

## Usage

- Watch for changes: `npm run watch`
- Build: `npm run build`
